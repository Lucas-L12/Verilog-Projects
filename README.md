# Verilog Projects

A collection of small Verilog projects for learning digital design concepts.

## Fibonacci FSMD

Implementation of a Fibonacci calculator using a **Finite State Machine with Datapath (FSMD)** architecture in Verilog.

Features:
- FSM with three states: `idle`, `op`, `done`
- iterative Fibonacci computation
- testbench for simulation in Vivado
- waveform verification
